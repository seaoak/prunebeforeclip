PruneBeforeClip

PruneBeforeClip は、
　　　　「Evernote などの Web Clipper で本文だけをクリップしたい！」
というニーズをちょっとだけ満たしてくれるブックマークレットです。

Evernote などで Web ページを保存しておくと、あとで検索できてとっても便利です。
でも、サイドバーの「過去の記事」リストとか、Google さんのテキスト広告とか、
本文以外のモノが検索にヒットしてしまって、ちょっと邪魔。
保存したい（＝あとで検索したい）のは本文だけ、ということがほとんどです。

ところで、先日、PrintWhatYouLike というサービスを知りました。
ブックマークレットとして実行するとページ上の要素を削除したりできるツールです。
これを使ってサイドバーとか広告とかを削除すれば、本文だけをクリップできます。

    参考）
        Evernote と PrintWhatYouLike の組み合わせが便利すぎる！
          -- マインドマップ１年生 plus ライフハック！


しかししかし、メジャーなニュースサイトの記事を毎日のようにクリップしていると、
いつもいつも同じように広告を削除するのがだんだん面倒になってきました・・・

で、同じ作業を繰り返しているなら自動化しちゃえ、というわけで、
作ってみたのがこの PruneBeforeClip というブックマークレット。

厳密に言うと「ブックマークレット」ではないのかもしれません。
ブックマークレットは JavaScript ファイルの読み込みタグを
ページに埋め込んでいるだけで、実際のページ要素の削除は
その JavaScript プログラムで行っています。

ちなみに、削除する要素は Seaoak の独断と偏見に基づいて指定されています。
要望は受け付けますが、実際に反映される保証はありません・・・

どうぞよろしく！

使い方

次のリンクをブックマークバーなどにドラッグ＆ドロップしてください：

    PruneBeforeClip

あとは、クリップしたいページでその項目をクリックすれば本文だけになります。
その状態で Evernote などの Web Clipper を実行すれば OK.

ちなみに、開発版（Beta バージョン）もあります。
次のリンクをブックマークして使えば開発版になります。

    PruneBeforeClip(dev)

開発版は、対応サイトの数や、対応サイトの HTML の変更への追従は早いですが、
もしかすると挙動があやしくなることがあるかもしれません。
開発版がうまく動かないときは通常版ブックマークレットをお使いください。

さらに不安定な Canary バージョンもあります。

    PruneBeforeClip(canary)

ソースコード

ブックマークレットで呼び出している JavaScript ファイルは次のものです：

    通常版： bookmarklet_01_00_00_00.js
    Beta版： bookmarklet_01_00_00_99.js
    Canary： bookmarklet_01_01_00_99.js

中身が気になるヒトは見てみてください。

ちなみに、MIT ライセンス（X11 ライセンス）なので、改造もご自由にどうぞ。
